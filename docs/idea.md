# NoteFlow: Idea y Propósito

## Problema
El usuario necesita un lugar centralizado para gestionar notas rápidas, listas de tareas y capturar ideas, sin la complejidad de herramientas corporativas.

## Usuario Objetivo
Profesionales y estudiantes que buscan una herramienta multiplataforma, rápida y con una interfaz limpia estilo "Microsoft To Do" que permita organizar el día a día.

## Funcionalidades principales
1. **Notas de texto**: Notas simples con título y cuerpo.
2. **Checklists**: Listas de tareas con estado de completado.
3. **Ideas**: Notas rápidas con sistema de etiquetas y colores.

## Funcionalidades futuras
- Sincronización en la nube con Firebase.
- Soporte para adjuntar imágenes.
- Categorización avanzada mediante carpetas.
