# Gestión del Proyecto: NoteFlow

## Tablero de Control
Estamos utilizando una metodología Kanban para organizar el desarrollo de NoteFlow.

## Columnas del Tablero
- **Backlog**: Funcionalidades futuras (ej. Sincronización en la nube, Archivo de notas).
- **Todo**: Tareas planificadas para la iteración actual.
- **In Progress**: Tareas en desarrollo activo.
- **Review**: Código pendiente de revisión o pruebas.
- **Done**: Funcionalidades finalizadas y verificadas.

## Organización
Las tareas están desglosadas en subtareas técnicas. A medida que una subtarea se completa, se mueve la tarjeta principal a la siguiente columna.
